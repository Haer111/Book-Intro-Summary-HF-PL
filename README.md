### 2. 주제 선정 이유
평소에 다음으로 읽거나 구매할 책을 고를 때 교보문고 베스트셀러를 주로 참고한다. <br>
흥미로워 보이는 제목의 책이 있으면 책 소개를 읽어보고 고르는데, 숏폼 시대에 이 책 소개마저도 너무 길다는 생각이 종종 들었다. <br>
그래서 책 소개를 요약하기로 했다. 책 소개가 이미 책을 요약한 것이니, 어찌보면 요약을 요약하는 것이다! 😉
